# 🧰 便利ツール オールインワン

開発者・クリエイター向けの Chrome 拡張機能です。日常的によく使うツールをポップアップからワンクリックで利用できます。

## 機能一覧

### ✏️ 文字列生成 / 文字数カウント
- **ランダム文字列生成** — ひらがな・カタカナ・英大文字・英小文字・数字・記号・半角カナから文字種を選択し、1〜10,000 文字のランダム文字列を生成
- **文字数カウント** — 入力テキストの文字数・UTF-8 バイト数・行数をリアルタイム表示

### ⏱️ エポック変換
- 現在のエポックミリ秒を表示（ボタンで手動更新）
- エポックミリ秒 → 日時（ローカル / UTC）変換
- 日時 → エポックミリ秒変換

### 🎨 カラー変換
- カラーピッカーおよび HEX 入力によるカラー指定
- HEX / RGB / HSL / Decimal RGB の相互変換・コピー
- スポイト機能による画面上のカラーサンプリング

### 🌐 翻訳
- 日本語 ↔ 英語の双方向翻訳
- Google Translate API を使用
- ワンクリックで言語方向を切り替え

### 🔤 正規表現
- 正規表現とテスト文字列を入力し、一致箇所をリアルタイムにハイライト表示
- 判定は常に `g` / `i` / `m`（全件・大文字小文字を区別しない・複数行）で実行
- 不正な正規表現はエラー文言を表示
- ヘルプアイコンから正規表現チートシート（メタ文字・量指定子・文字クラス等）をモーダルで参照

### *️⃣ 正規表現生成
- 長さ条件（最小・最大）と許可する文字種（英大文字・英小文字・数字・記号）を指定し、バリデーション用の正規表現を生成
- 文字種ごとに「1文字以上含む」必須条件を設定可能（先読みアサーションを自動生成）
- 生成した正規表現に対してテスト文字列を入力し、合格・不合格を判定

### 🔎 サイト内検索
- 正規表現でページ内テキストを検索し、一致箇所をページ上にハイライト表示（DOM を書き換えない CSS Custom Highlight API を使用）
- 実行は **Enter またはボタン**（リアルタイム判定はしない）。「**正規表現**」チェックボックス（既定 ON。OFF で入力をそのままの文字列として検索）と「**大文字小文字区別**」チェックボックス（既定オフ）
- ヒット件数・現在位置（`n / 総数`）を表示。ポップアップ側の結果リスト（前後の文脈付き）をクリックするとその箇所へジャンプ
- **ページに検索バーを表示** — ポップアップはフォーカスが外れると閉じるため、現在のページに常駐する検索バー（Shadow DOM で隔離した Ctrl+F 風の小バー）を注入できる。ポップアップを閉じても・タブを切り替えても残り、入力／`.*`／`Aa`／▲▼（Enter=次、Shift+Enter=前、Esc=閉じる）でその場で検索を続けられる
- 「**全タブ検索**」チェックボックスで現在のページ／全タブを切替。全タブは**タブ別（ファビコン＋タイトル＋件数の見出し）に一覧**し、見出しやスニペットのクリックでそのタブを開いてハイライト
- 検索ワードを履歴として保存し、入力補完で再利用。直近の検索結果（リスト・件数・現在位置）も保存し、ポップアップを開き直すと復元（`chrome.storage.local`）
- ※ ページ操作のため `scripting` 権限と全サイトへの `host_permissions` を使用。`chrome://`・Web ストア・PDF などスクリプトを注入できないページ、および iframe 内は対象外。一致数が多い場合は上限で打ち切る

### 🔢 電卓
- Windows 標準電卓と同じ**逐次計算方式**のシンプルな電卓（演算子を押すたびに直前までを計算、優先順位なし）
- 加算・減算・乗算・除算・剰余（`%`）に対応
- ボタン UI とキーボードの両方で入力可能（`0-9 .`、`+ - * / %`、`Enter`/`=`、`Backspace`、`Escape`=全消去、`Delete`=直前入力消去）
- 数値のクリップボード貼り付けに対応（数値以外は無視）
- 整数は多倍長（`BigInt`）で正確に計算。入力は最大 16 桁、計算結果は最大 20 桁まで（絶対値）で、結果がこれを超えると `Overflow`、0 除算は `Cannot divide by 0` と表示
- 計算履歴（最大 20 件）を一覧表示し、クリックで結果を再利用。履歴は `chrome.storage.local` に保存され、ポップアップを開き直しても残る

### 📝 メモ
- 拡張機能のストレージ（`chrome.storage.local`）に自動保存されるメモ帳
- 入力 200ms 後にデバウンス保存

### ℱ フォント取得
- ページ上の要素をクリックしてフォント情報を取得するインスペクター
- 要素のホバーでハイライト表示、クリックで font-family / font-size / font-weight / font-style / color を取得
- 結果はページ上の固定パネルに表示。各プロパティの個別コピーに対応。↗ ボタンでポップアップを開き直し、取得した値をポップアップに反映
- ポップアップ側でフォント設定を手動入力し、サンプルテキストでプレビュー表示
- ホイールスクロールでページスクロール可能。Esc または閉じるボタンで終了
- ※ 初期状態ではタブ非表示。設定の「表示するツール」から有効化。`chrome://` や拡張機能ページ・PDF などスクリプトを注入できないページは対象外

### ⚙️ 設定
- ヘッダー右の ⚙️ から開く設定オーバーレイ（左のナビで切り替えるタブ構成）で、以下を管理
- **表示するツール** — タブに表示するツールをスイッチで選択し、ドラッグで並び替え。変更は即時反映（すべて非表示にもできる）
- **入力状態の保持** — 各ツールの入力・結果の自動保存／復元を、全体（マスター）とツールごとに切り替え（全体オフで一括停止）
- **ストレージ** — `chrome.storage.local` の使用量を可視化（合計＋上限約 10MB に対する使用率バー＋ツール別内訳）し、ツール単位／一括でデータを消去（実行前に確認ダイアログ）

### 💾 状態の保持
- 各タブの入力値・変換結果を拡張機能のストレージ（`chrome.storage.local`）に自動保存し、ポップアップを開き直すと前回の作業内容が復元される
- 最後に開いていたタブも記憶し、次回はそのタブが開いた状態で起動する

## インストール

1. このリポジトリをクローンまたはダウンロード
   ```
   git clone <repository-url>
   ```
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパー モード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択
