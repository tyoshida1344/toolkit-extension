# 🧰 便利ツール オールインワン

開発者・クリエイター向けの Chrome 拡張機能です。日常的によく使うツールをポップアップからワンクリックで利用できます。

## 機能一覧

### ✏️ 文字列生成 / 文字数カウント
- **ランダム文字列生成** — ひらがな・カタカナ・英大文字・英小文字・数字・記号・半角カナから文字種を選択し、1〜10,000 文字のランダム文字列を生成
- **文字数カウント** — 入力テキストの文字数（書記素単位）・UTF-8 バイト数・行数をリアルタイム表示

### ⏱️ エポック変換
- 現在のエポックミリ秒をリアルタイム表示
- エポックミリ秒 → 日時（ローカル / UTC）変換
- 日時 → エポックミリ秒変換

### 🎨 カラー変換
- カラーピッカーおよび HEX 入力によるカラー指定
- HEX / RGB / HSL / Decimal RGB の相互変換・コピー
- EyeDropper API による画面上のカラーサンプリング（Chrome / Edge）

### 🌐 翻訳
- 日本語 ↔ 英語の双方向翻訳
- Google Translate API を使用
- ワンクリックで言語方向を切り替え

### 📝 メモ
- ブラウザのローカルストレージに自動保存されるメモ帳
- 入力 500ms 後にデバウンス保存・保存状態を表示

## インストール

1. このリポジトリをクローンまたはダウンロード
   ```
   git clone <repository-url>
   ```
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパー モード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択
