カラー変換のスポイト機能を強化し、安定性を改善しました。

### Added
- カラーピッカーでスポイト選択時にポップアップが自動で閉じるように改善
- サイト内検索のミニポップアップからポップアップを開き直せる機能を追加

### Changed
- EyeDropper API 非対応の環境でもスポイトが使える代替方式（captureVisibleTab）に対応
- エラー時のトースト表示を統一し、エラースタイルを追加
- カラー変換の出力初期値を自動生成に変更し、初回表示の一貫性を向上
- 文字列生成の数値入力で最大値を超えた値が即時クランプされるように改善
- 各種パフォーマンス改善（デバウンス追加、タブ切替最適化、ストレージ一括取得）

### Fixed
- カラーピッカー操作時にイベントループが発生しブラウザがフリーズする問題を修正
- スクロール中にスポイトで取得する色の位置がずれる問題を修正
- サイト内検索バーにフォーカス中、ページ側のショートカットキーが誤動作する問題を修正
